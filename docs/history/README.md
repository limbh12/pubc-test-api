- `2025-11-06_JDK18업그레이드.md`

## 📝 이력 파일 작성 가이드

각 이력 파일은 다음 내용을 포함해야 합니다:

### 1. 개요
- 작업 목적
- 주요 변경 사항 요약

### 2. 상세 변경 내용
- 변경된 파일 목록
- 이동/삭제/추가된 항목
- 코드 변경 사항

### 3. 변경 이유
- 왜 이 작업이 필요했는지
- 어떤 문제를 해결하는지

### 4. 영향 범위
- 영향받는 시스템/모듈
- 관련된 문서

### 5. 검증 방법
- 변경 사항 확인 방법
- 테스트 결과

## ⚠️ 주의사항

1. **현재 문서 업데이트 금지**
   - 이 디렉토리의 파일은 수정하지 않습니다
   - 새로운 변경사항은 새 파일로 추가합니다

2. **민감 정보 제외**
   - 비밀번호, API 키 등 민감 정보 포함 금지
   - 개인정보 포함 금지

3. **명확한 문서화**
   - 미래의 개발자가 이해할 수 있도록 명확하게 작성
   - 스크린샷이나 다이어그램 적극 활용
