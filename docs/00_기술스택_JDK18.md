## ✅ PUBC 모듈 호환성

### iros_pubc_1.1.jar
- **컴파일 타겟**: Java 5 (1.5)
- **JDK 1.8 호환**: ✅ **완벽 호환**
- **근거**: Java 하위 호환성 (Java 5 바이트코드 → Java 8 JVM 실행 가능)

### PUBC 의존 라이브러리
| 라이브러리 | PUBC 버전 | Java 8 호환 |
|-----------|----------|------------|
| Spring Framework | 2.5.6 | ✅ 완벽 호환 |
| iBATIS | 2.3.4 | ✅ 완벽 호환 |
| eGovFramework | 1.0.0 | ✅ 호환 |

### 호환성 전략
```
프로젝트 코드 (Spring 4.3.30)
    ↓
PUBC 모듈 (Spring 2.5.6 내장)
    ↓
Spring 4.x가 2.5.6 API 하위 호환
    ↓
✅ 정상 동작
```

## 🆕 Java 8 활용 기능

### Lambda Expression
```java
// 사용자 필터링
List<UserVO> activeUsers = users.stream()
    .filter(user -> "ACTIVE".equals(user.getStatus()))
    .collect(Collectors.toList());
```

### Stream API
```java
// 사용자명 추출
List<String> names = users.stream()
    .map(UserVO::getUserName)
    .collect(Collectors.toList());
```

### Optional
```java
// NullPointerException 방지
Optional<UserVO> user = Optional.ofNullable(userService.getUser(userId));
UserVO result = user.orElseThrow(() -> new NotFoundException("사용자 없음"));
```

### Date/Time API
```java
// java.time 패키지 사용
LocalDateTime now = LocalDateTime.now();
LocalDate today = LocalDate.now();
```

## 📦 SQL Mapper 이중 구조

### MyBatis (프로젝트 코드용)
```xml
<!-- SqlSessionFactory -->
<bean id="sqlSessionFactory" class="org.mybatis.spring.SqlSessionFactoryBean">
    <property name="dataSource" ref="dataSource"/>
    <property name="configLocation" value="classpath:mybatis-config.xml"/>
</bean>

<!-- Mapper Scan -->
<bean class="org.mybatis.spring.mapper.MapperScannerConfigurer">
    <property name="basePackage" value="iros.test.user.dao"/>
</bean>
```

### iBATIS (PUBC 모듈용)
```xml
<!-- SqlMapClient (PUBC 전용) -->
<bean id="pubcSqlMapClient" class="org.springframework.orm.ibatis.SqlMapClientFactoryBean">
    <property name="configLocation" value="classpath:iros/pubc/sqlmap/sql-map-config-pubc.xml"/>
    <property name="dataSource" ref="dataSource"/>
</bean>
```

**장점**:
- ✅ 프로젝트: MyBatis 3.x 최신 기능 활용
- ✅ PUBC: iBATIS 2.x 그대로 유지
- ✅ 동일 DataSource 공유
- ✅ 네임스페이스 분리로 충돌 없음

## 📌 주의사항

### DO
- ✅ JDK 1.8 사용
- ✅ Spring 4.3.30 사용
- ✅ MyBatis 3.5.13 사용 (프로젝트 코드)
- ✅ Java 8 기능 적극 활용 (Lambda, Stream)
- ✅ PUBC JAR 파일 `lib/` 디렉토리에 배치
- ✅ pom.xml에 `<systemPath>` 설정

### DON'T
- ❌ JDK 1.6 사용 금지 (레거시)
- ❌ Spring 2.5.6 직접 사용 금지 (PUBC만 사용)
- ❌ iBATIS 프로젝트 코드에 사용 금지 (PUBC만 사용)
- ❌ PUBC JAR 파일 수정 금지
- ❌ Java 9+ 전용 기능 사용 금지 (호환성)
